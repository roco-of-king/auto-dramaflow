# 电影声音设计基础

> 声音如何服务故事——从心理声学到叙事功能的完整声音设计方法论。Toonflow 项目此前完全缺失的领域。

---

## 一、声画关系分类体系

### 四类声音

电影声音按与画面和故事世界的关系分为四类：

| 分类 | 定义 | 叙事功能 | 典型示例 |
|------|------|----------|----------|
| **画内声音（On-screen）** | 声源在画面中可见 | 最直接的叙事信息 | 嘴唇同步对白、可见的关门 |
| **画外声音（Off-screen）** | 声源在故事空间中但画面外 | 制造好奇/恐惧/氛围 | 画外喊叫、不可见的收音机、环境声 |
| **非剧情声音（Non-diegetic）** | 故事人物听不到 | 引导观众情绪 | 配乐、画外音旁白、夸张音效 |
| **幻听（Acousmatic）** | 听到声音但看不到声源 | 制造神秘感 | 隐藏声源，保留揭示悬念 |

### 关键操作

- **画外声音**分两种：主动（引发好奇——那是什么声音？）和被动（营造氛围——环境声）
- **去幻听（De-acousmatic）**：揭示声源的戏剧时刻——声音从"幻听"变为"画内"，产生巨大冲击力（如《绿野仙踪》拉开帘子）
- 画外声音过多 → 观众不知道该场景空间的边界在哪里
- 画外声音 ≠ 非剧情声音：画外声音属于故事世界（角色能听到），非剧情声音不属于故事世界

## 二、听觉层级体系

### 三层注意力模型

| 层级 | 注意力状态 | 内容 | 叙事角色 |
|------|-----------|------|----------|
| **前景层（Figure）** | 有意识关注 | 对白、关键音效 | "音型"——故事焦点 |
| **中景层（Midground）** | 听到但不直接注意 | 环境声、辅助音效 | "辅助"——空间感 |
| **背景层（Background）** | 影响潜意识 | 低频氛围、极轻微环境声 | "底色"——情绪基调 |

### 动态层级转换

层级之间可以相互转换——一个背景声音可以突然跳到前景（如远处逐渐接近的警笛），也可以从前景退到背景（如对话结束后环境声重新"显现"）。

**核心规则**：
- 语言通常为最高层级，音乐和音效需围绕语言调整
- 同一时刻只有一个声音"说话"——前景层只能有一个主导元素
- 层级转换必须有叙事铺垫（角色转头、摄影机移动、情节推进）
- 掩蔽效应：低频声音可以"掩蔽"高频声音，利用这一效应可建立和打破层级

### AI 生成应用

```
audio mix: dialogue in foreground (crisp, intimate), 
ambient city sounds in midground (muffled, distant), 
low-frequency room tone in background (subconscious, barely perceptible)
```

## 三、四层声能转化模型

### 核心框架

声音设计必须在四个平行层面同时工作，只关注技术层面是"声音匠人"而非"声音设计师"的区别。

| 层面 | 传导链 | 关注点 | 诊断问题 |
|------|--------|--------|----------|
| **物理层** | 声源→录制→混录→还音→听觉系统 | 技术质量 | 声音录得干净吗？ |
| **情绪层** | 故事→人物→冲突→音调→音乐→节奏→观众反应 | 戏剧效果 | 声音让观众感受到什么？ |
| **智力层** | 作曲→语义思想→悬念→幽默→意识提升 | 美学意义 | 声音传达了什么样的思想？ |
| **道德/精神层** | 困境→关系→鉴定→内心质疑 | 深层价值 | 声音引导观众做什么样的价值判断？ |

### 使用场景

- 当场景"感觉不对"但说不清原因时 → 逐层检查哪一层断裂了
- 物理层完美但情绪层缺失 = 技术上完美但观众无感
- 情绪层强烈但物理层粗糙 = 情感到位但出戏
- 层间对齐检查：物理层的声音选择是否支持情绪层的目标？

## 四、聆听模式分类

### 三种聆听方式

| 模式 | 描述 | 适用场景 |
|------|------|----------|
| **简化聆听（Reduced Listening）** | 只关注声音本身的属性（音高、音色、节奏），忽略声源和意义 | 声音设计细节调整 |
| **因果聆听（Causal Listening）** | 通过声音识别声源——"这是什么声音？" | 悬疑、揭示、空间建立 |
| **语义聆听（Semantic Listening）** | 解读声音传递的语言/符号意义 | 对白、代码化的音效（如门铃、警报） |

观众同时在三种模式间切换。声音设计师需要控制这种切换——引导观众何时"因果聆听"（寻找声源），何时"语义聆听"（理解信息）。

## 五、声音引导法（Entrainment）

### 核心原理

听众的生理状态可以被声音的节奏和频率**同步引导**——心跳、呼吸、脑波会无意识地与声音节奏同步。

### 操作策略

| 目标 | 声音策略 |
|------|----------|
| 增强紧张 | 逐渐加速的节奏 + 升调 + 增大音量 |
| 释放紧张 | 从快速节奏突然切换到慢速或静默 |
| 催眠/入神 | 稳定的低频脉冲 + 重复循环 |
| 唤醒/刺激 | 突然的音量峰值 + 不谐和音 |

### AI 视频应用

```
tension building: heartbeat-like low-frequency pulse gradually accelerating, 
layered with rising string dissonance, reaching peak then abruptly cutting to silence
```

## 六、对比戏剧法与声音能量模型

### 对比戏剧法

戏剧效果来自声音的**对比**而非绝对值：
- 从极吵到极静 = 震撼
- 从极静到极吵 = 惊吓
- 从有序到混乱 = 失控
- 从混乱到有序 = 控制

持续的同质声音会让观众适应并忽视——就像持续的白噪音。变化的对比才是戏剧。

### 声音能量密度

用声音的能量密度控制场景张力：
- **高能量密度**：多层声音同时存在、高频成分多、音量变化大 → 紧张/混乱
- **低能量密度**：单层或少层声音、低频为主、音量稳定 → 平静/沉思
- **能量密度曲线**：场景的张力弧线应该反映在声音能量密度变化中

## 七、声音设计检查清单

### AI 视频/动画声音设计清单

- [ ] 每个声音元素是否明确标注为画内/画外/非剧情/幻听？
- [ ] 前景/中景/背景三层是否分配清晰？同一时刻只有一种声音在前景？
- [ ] 四层声能模型是否每层都经过检查？
- [ ] 是否有"去幻听"的戏剧时刻设计？
- [ ] 是否有声音对比（吵/静、有序/混乱）？
- [ ] 声音能量密度曲线是否跟随场景张力弧线？
- [ ] 是否利用了静默作为主动创作选择（而非"没有声音"）？
- [ ] 关键对白是否在任何层级转换中都不被掩蔽？

### 常见失败模式

- **所有声音都在前景** → 观众无法分辨重点
- **层级转换无铺垫** → 观众被"拽出"故事
- **过度关注物理层** → 技术上完美但情感上空洞
- **画外声音过度使用** → 制造"廉价恐惧"，透支悬念功能
- **忽略静默的力量** → 把静默当"没有选择"而非主动决策
