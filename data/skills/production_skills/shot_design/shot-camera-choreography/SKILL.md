# 编舞式镜头设计

## R — 原文 (Reading)
> 拍电影就像编舞，每一个场景，你都要设计好演员和摄像师的舞步。
>
> — Christopher Kenworthy, 前言, p.3

---

## I — 方法论骨架 (Interpretation)

编舞法是全书100个技法的元框架——每个技法本质上都是"演员舞步+摄影机舞步"的一种特定组合。核心思想是：演员的运动路线和摄影机的运动轨迹必须事先设计并精确配合，不是现场即兴。

这套方法论的价值在于把"感觉性"的镜头指导变成"可规划"的编舞设计。它提供四步操作流程：(1) 设计演员的运动路线（从哪里到哪里，为什么动）；(2) 设计摄影机的运动轨迹（跟拍、推近、后退、转向）；(3) 确保两者在时间上同步——演员到达某个位置时摄影机也到达对应位置；(4) 排练至流畅，任何不同步都会暴露摄影机的存在。

关键原则：摄影机运动必须有叙事理由。演员转身时摄影机配合转向（表达决心），演员静止时摄影机也静止（表达等待），演员和摄影机同时转向制造动感对比（表达危险）。没有叙事目的的摄影机运动是噪音。

## A1 — 书中的应用 (Past Application)

### 案例 1: 《锅盖头》的人机同时转向（c25）
- **问题**: 如何用镜头表现角色从犹豫到下定决心的心理变化？
- **方法论的使用**: 先拍一段平淡画面（演员迎着摄影机跑来），然后演员超过摄影机后转身向后看，摄影机同时向演员运动并调头。演员和摄影机同时转向的动感对比制造强烈戏剧性。先静后动的切换外化了内心决定。
- **结论**: 一动一静的结合比持续运动更有戏剧性——静止是蓄力，运动是释放。
- **结果**: 演员和摄影机同时转向的动感对比让观众深刻感受到危险近在咫尺。

### 案例 2: 《狼的时刻》的十字交叉争执（c30）
- **问题**: 争执场景中如何让两个角色的冲突看起来自然而不做作？
- **方法论的使用**: 两人做十字交叉行走，方向不断变化。走开的演员来到镜头前（近处），另一人在后面用短焦镜头拍摄（显得距离远但靠近快）。绕弧线走+方向变化比直线前后走动更自然。
- **结论**: 编舞法不仅设计运动方向，还设计运动节奏——方向变化本身就是冲突的外化。
- **结果**: 十字交叉路线让争执场景看起来真实而不做作。

### 案例 3: 《指环王》的角色决定出发（c24）
- **问题**: 如何用镜头告诉观众"角色做了一个重大决定，要起程了"？
- **方法论的使用**: 摄影机迎着演员推进、摇拍，超过演员后调头。观众先看到近处特写（内心），随后看到他走向开阔场地（行动）。摄影机超过演员后继续后退，把视野拉宽，表现演员融入新环境。
- **结论**: 摄影机的运动轨迹就是叙事线——从"进入内心"到"走向世界"。
- **结果**: 摄影机运动配合演员起步，清楚告诉观众"他做了重大决定"。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 用户要拍一个有复杂运动的场景，不知道演员和摄影机怎么配合——"演员走位和摄影机怎么协调"
2. 用户要表现角色的心理变化（犹豫→决定、平静→激动），想通过运动设计外化内心——"怎么用镜头表现角色下定决心"
3. 用户要拍争执/对话场景，不想用死板的正反打——"不想用过肩镜头拍对话，有什么别的办法"
4. 用户拍完发现"摄影机存在感太强"，观众出戏——"怎么让摄影机运动不被注意到"
5. 用户要为一个场景设计完整的镜头方案——"帮我设计这场戏的镜头"

### 与相邻 skill 的区分

- 与 `shot-telephoto-compression` 的区别: 长焦特效是编舞法的一种特定技术组合（用长焦焦距实现特定错觉），编舞法是整体设计方法论。编舞法回答"怎么设计整体运动"，长焦特效回答"怎么用镜头焦距制造错觉"。
- 与 `shot-still-vs-motion-decision` 的区别: 静止vs运动是编舞法中"什么时候让演员动、什么时候让演员停"的决策工具，编舞法是包含所有运动组合的完整框架。
- 与 `shot-dual-camera-tension` 的区别: 双机位对比是编舞法在双摄影机场景中的特殊应用，编舞法覆盖单机位和多机位的所有情况。

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **设计演员的运动路线**
   - 完成标准: 画出演员在场景中的起点、终点和运动路径。每个运动转折点都有叙事理由（去拿东西、被吸引、决定改变方向）。能回答"演员为什么要从A走到B？"
   - 判停条件: 如果演员的运动没有叙事动机（只是为了"走一走"），需删除该运动或赋予理由。

2. **设计摄影机的运动轨迹**
   - 完成标准: 摄影机的每个运动（跟拍、推近、后退、转向）都对应演员的一个动作或情绪变化。能回答"摄影机为什么在这个时候动？"运动轨迹和演员路线在时间轴上对齐。
   - 判停条件: 如果摄影机运动先于或后于演员动作（不同步），需调整时机。参考反例 ce20：摄影机停止时演员还在动=暴露摄影机存在。

3. **排练验证同步性**
   - 完成标准: 演员和摄影机的运动在时间上完全同步——演员到达某位置时摄影机也到达对应位置。任何不同步都会打破"第四面墙"。排练至少3次直到流畅。
   - 判停条件: 如果多次排练仍无法同步，考虑简化运动方案（减少运动点或降低运动速度）。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 场景本身不需要运动（如静态对话、独白场景），编舞法不适用于"摄影机该不该动"的判断
- 用户只想了解某种摄影机运动的技术操作（如"怎么用稳定器跟拍"），不涉及演员配合
- 纪录片或真实场景拍摄——编舞法假设可以事先设计，不适用于不可控的真实场景

### 作者在书中警告的失败模式

- **摄影机停止时演员还在动（ce20）**: 演员已经停下来了摄影机还在后退，观众就会发现摄影机的存在。演员和摄影机必须同时开始、同时停止。
- **没有理由的演员移动（ce13）**: 千万不要让演员乱逛。每个移动都必须有叙事理由，无目的的移动让观众意识到"这是在拍电影"。
- **为移动摄影机而乱动摄影机（ce04）**: 两种极端——摄影机不停移动但没有叙事目的（眩晕），或完全不动（沉闷）。移动本身不是目的，叙事才是。

### 作者的盲点 / 时代局限

- 编舞法假设导演有充分的排练时间，对快节奏低成本制作的适用性未讨论。
- 对手持摄影的即兴风格（如达内兄弟、保罗·格林格拉斯）未做探讨——这类风格恰恰以"不事先设计"为特征。
- 未涉及数字时代的虚拟摄影机运动（如无人机、FPV穿越机）对编舞法的挑战。

### 容易混淆的邻近方法论

- 舞台编排（Blocking）：舞台编排关注演员在舞台上的位置关系，编舞法关注演员和摄影机的双重运动配合。
- 分镜头脚本（Storyboard）：分镜头是静态的画面规划，编舞法是动态的运动规划。两者互补但不等同。
