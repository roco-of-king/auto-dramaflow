# 记忆色彩三大要素校准法

## R — 原文 (Reading)

> 记忆色彩三大要素：肤色、天空、植物。观众对这三种颜色有强烈的记忆预期——如果偏离太多，画面会'看起来不对'。
>
> — Van Hurkman, 第8章, p.g_0000

---

## I — 方法论骨架 (Interpretation)

观众对三种颜色有强烈的记忆预期：肤色（所有人种在矢量示波器上落在同一狭窄楔形内）、天空（蓝色，不是紫色或绿色）、植物（绿色，不是黄色或蓝色）。这三种颜色如果偏离观众的记忆预期，画面会'看起来不对'，即使技术上是准确的。

独特价值：记忆色彩比实际色彩更饱和——研究显示观众记忆中的色彩比实际约饱和0.95孟塞尔色度值。这意味着调色师需要将记忆色彩调整到比实际更饱和的水平，才符合观众的心理预期。

---

## A1 — 书中的应用 (Past Application)

### 案例 1
- **问题**: 如何在实际调色中应用此原理
- **方法论的使用**: 纪录片调色师发现，将天空调整到技术准确的蓝色后，观众觉得'不够蓝'。原因：观众记忆中的天空比实际更蓝。解决方案：将天空的饱和度略微提高到记忆色彩水平，而非技术准确水平。
- **结论**: 记忆色彩的容忍度极低——天空偏绿 5% 观众就能感知到，但衣服偏色 15% 可能没人注意。优先校准天空和肤色
- **结果**: 天空和肤色回到观众心理预期的"正确"色域，画面整体看起来"舒服"而非"技术准确但违和"

### 案例 2: 《指环王》中土世界植被校色
- **问题**: 新西兰实景拍摄的绿色植被在调色后偏黄，观众反馈'看起来不像中土世界'
- **方法论的使用**: 调色师发现植被色彩被调整到了技术准确的绿色，但观众记忆中的森林比实际更绿更浓郁。将植被饱和度提高到记忆色彩水平（比技术值约高1个孟塞尔色度），同时保持天空和肤色在各自记忆色彩范围内
- **结论**: 三大记忆色彩（肤色、天空、植物）都需要超越技术准确值才符合观众心理预期
- **结果**: 植被看起来更符合观众对'魔法森林'的想象，画面整体更自然

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **检查三大记忆色彩**
   - 完成标准: 用矢量示波器检查肤色是否在楔形内、天空是否偏蓝、植物是否偏绿。完成标准：三种记忆色彩都在观众预期范围内

2. **与技术准确值对比**
   - 完成标准: 记忆色彩比技术准确值更饱和。完成标准：将色彩调整到记忆色彩水平而非技术准确水平
   - 判停条件: 若条件不满足，先确认素材质量和监视器校准

3. **检查整体平衡**
   - 完成标准: 三种记忆色彩的相对关系要正确。完成标准：肤色、天空、植物的色彩关系协调

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 不要只关注肤色而忽略天空和植物——三者需要协调
- 不要将记忆色彩调整到技术准确值——观众会觉得'不够'

### 作者在书中警告的失败模式

- 只校准肤色但天空偏紫——观众会觉得画面'不对'但说不出为什么
- 将记忆色彩调整到技术准确水平——观众觉得'不够饱和'

### 作者的盲点

作者的记忆色彩研究主要基于西方观众，不同文化对记忆色彩的预期可能有差异。此外，数字摄影机的色彩科学可能已经预校正了记忆色彩。
