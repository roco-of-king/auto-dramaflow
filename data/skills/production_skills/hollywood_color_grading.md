# 好莱坞电影调色系统

---

## 一、专业调色核心工作流

### 1. 一级校色三步法

一级校色是所有调色工作的基础，目的是建立技术正确的画面。核心流程为三步迭代：

| 步骤 | 操作 | 工具 | 判停标准 |
|------|------|------|----------|
| **1. 对比度优先** | 黑位 → 0-5%，白位 → 90-100%，中间调分布合理 | Lift/Gamma/Gain | 画面有"密度"和"活力" |
| **2. 色彩平衡** | 校正偏色，按暗部→中间调→高光顺序逐一平衡 | 三路色彩校正 / Offset / RGB曲线 | 灰色/白色物体看起来中性 |
| **3. 饱和度微调** | 检查并调整整体饱和度 | 矢量示波器 + 肉眼 | 色彩鲜艳但不过度，肤色健康 |

**关键认知**：这三步不是线性的而是迭代的——调整对比度会改变饱和度（RGB处理下扩大对比度自动增加饱和度），调整色彩平衡会改变对比度感知。通常3轮迭代即可收敛。

### 2. 先对比度后色彩的调整顺序

**为什么必须先调对比度？** 色彩平衡控件的影调分区（暗部/中间调/高光）依赖于亮度通道。改变对比度会改变各影调区的范围——如果先调色彩再调对比度，之前精心调整的色彩平衡会被打乱。

**例外**：严重偏色的素材 → 先用 Offset 整体校正，再调对比度。

### 3. Log 素材调色流程

对于 Log 编码素材（S-Log、V-Log、C-Log、BRAW等），使用特殊流程：

1. **在 Log 状态下做整体调整**（一级校色的大部分工作在正常化之前完成）
2. **正常化（LUT / Color Space Transform）**
3. **在正常化状态下做细节调整**

## 二、记忆色彩三大要素校准

### 核心发现

观众对三种颜色有最强的记忆预期——**肤色、天空、植物**。如果这三种颜色偏离观众的记忆预期，画面会"看起来不对"，即使技术上是准确的。

**关键数据**：观众记忆中的色彩比实际色彩约饱和 0.95 孟塞尔色度值。因此需要将记忆色彩调整到比实际**更饱和**的水平，才符合观众心理预期。

### 三大记忆色彩容忍度

| 记忆色彩 | 容忍度 | 常见偏色问题 | 校准参考 |
|----------|--------|-------------|----------|
| **肤色** | 极低（±5°色相） | 偏绿、偏黄、偏红 | 矢量示波器 I-bar 附近楔形范围 |
| **天空** | 极低（偏绿5%观众即察觉） | 偏紫、偏绿 | 蓝色色相区，避开紫色/绿色 |
| **植物** | 中（偏黄最明显） | 偏黄（枯萎感）、偏蓝（不自然） | 绿色色相区，偏向黄绿而非蓝绿 |

### 校准原则

- 优先校准天空和肤色（容忍度最低）
- 记忆色彩比技术准确值更饱和才符合观众预期
- 三种记忆色彩需要协调——只校准肤色但天空偏紫会让观众觉得"不对"

## 三、肤色矢量示波器基准

### 核心原理

所有人类肤色——无论种族——在矢量示波器上都落在围绕 **I-bar（肤色线）** 的狭窄楔形范围内，色相差异通常不超过5度。

**物理原因**：肤色由黑色素（棕/黄色）、血红蛋白（红色）和β-胡萝卜素（橙色）三种物质决定，组合比例决定精确位置，但不显著改变色相。

### RGB 分量示波器快速判断

肤色在 RGB 分量示波器上始终表现为：**红色通道最强 > 绿色通道较弱 > 蓝色通道最弱**。如果这个顺序被打破 → 画面偏色。

### 操作步骤

1. 在矢量示波器上找到肤色波形（应在 I-bar 附近）
2. 判断偏色方向（偏 I-bar 左侧=偏黄/橙，偏右侧=偏红）
3. 用色彩平衡控件将肤色波形推回 I-bar 附近（保留不同人种的自然色相差异）
4. 在 RGB 分量示波器上确认红>绿>蓝的顺序

**警告**：不要把所有人种肤色都推到同一点——保留自然差异（约5°色相），否则肤色会"塑料化"。

## 四、镜头匹配六步法

### 目标

让同一场景内的所有镜头在播放时看起来**感知一致**——观众不会因为色彩跳动而出戏。

### 六步工作流

| 步骤 | 操作 | 注意事项 |
|------|------|----------|
| 1. 播放整个场景 | 整体感知差异，不急于动手 | 标记差异最大的镜头对 |
| 2. 选择参考镜头 | 选择光线最好、曝光最准的镜头 | 确保参考镜头的一级校色已完善 |
| 3. 匹配对比度 | 用波形监视器确认黑位/白位一致 | 先匹配骨架 |
| 4. 匹配色彩平衡 | 用分量示波器比较RGB通道对齐 | RGB三通道分别对齐 |
| 5. 独立评估饱和度 | 用肉眼 + 矢量示波器 | 最容易被忽略的维度 |
| 6. 排查异常元素 | 是否有分散注意力的鲜艳物体需单独处理 | 如气球、广告牌等 |

### 核心心法

- **"第二天规则"**：如果某个镜头困扰你超过5分钟 → 先跳过，第二天回来看，一两分钟就能解决
- **静止帧匹配≠播放匹配**：从头播放2-3次确认一致性
- **饱和度是最容易忽略的维度**：色彩平衡正确但饱和度不同时，容易被误判为色彩偏移

## 五、创意 Look 开发流程

### 核心原则

**技术校色与风格化处理必须分离**——分别放置在不同的操作节点/层中：
- 分离后一级校色可独立调整而不影响风格化
- 风格化可随时修改或去除而不破坏基础画面

### 开发流程

1. **评估与规划**：观察原始素材 → 确定风格方向（参考导演意图、影片类型、场景情绪）
2. **建立一级校色基础**：技术正确的画面
3. **二级分区**：用 HSL 选色/形状遮罩分离需要单独处理的区域（肤色/天空/暗部）
4. **叠加风格化**：着色、偏色、暗部色调、Vibrance、对比度曲线
5. **回检基础**：确认技术指标（肤色健康、信号合法）

### 风格化工具

| 工具 | 效果 | 典型用途 |
|------|------|----------|
| 着色（Tint） | 整体画面向某个色相偏移 | 复古暖调、冷调忧郁 |
| 暗部色调 | 阴影区域的色彩注入 | 橙蓝对比暗部（蓝）、胶片感 |
| Vibrance | 保护肤色的饱和度增强 | 自然增强而不让肤色过饱和 |
| 色彩对比曲线 | YRGB 独立曲线调整 | 交叉冲洗、日调夜 |

### AI 提示词应用

```
color grading: [一级校色描述] + [风格化方向]
"properly exposed with deep blacks and neutral color balance, 
then styled with warm golden highlights and cool blue shadows, 
skin tones protected and natural, filmic contrast curve"
```

## 七、调色参数维度

```
[色彩平衡] → 阴影/中间调/高光 各自颜色倾向
[对比度]   → 柔和(Soft) → 中 → 极高对比(High-key Contrast)
[饱和度]   → 灰暗(Muted) → 中 → 鲜艳(Vivid)
[光晕]     → Bloom/Glow 强度 (0.0-1.0)
[颗粒]     → 胶片颗粒百分比 (0-20%)
[暗角]     → Vignette 强度 (0.0-0.5)
[色差]     → Chromatic Aberration (轻微/关)
```

## 十、AI 调色检查清单

- [ ] 一级校色：对比度是否先于色彩调整？
- [ ] 记忆色彩：肤色、天空、植物是否在校准范围内？
- [ ] 肤色：RGB 分量示波器是否红>绿>蓝？
- [ ] 镜头匹配：同一场景镜头是否感知一致？
- [ ] 风格化：技术校色与创意风格是否分离？
- [ ] 广播安全：亮度和色度是否在合法范围？
