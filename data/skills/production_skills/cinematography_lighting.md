# 电影灯光与空间连续性

> 为 AI 视频/图像生成 Agent 提供的系统化灯光与空间连续性知识。

---

## 一、三点布光系统

### 核心原理

三点布光是电影摄影最基础的灯光拓扑结构，由三个功能明确的光源组成。核心价值不在于"三盏灯"，而在于将灯光分解为**三个独立可控的功能层**——每一层都可以独立调整强度、方向、品质和颜色。用一盏灯加反光板同样可以实现功能分离。

### 三灯功能

| 灯光 | 功能 | 关键决策 |
|------|------|----------|
| **主光（Key Light）** | 场景最主要光源，定义照明方向、阴影位置和整体明暗分布 | 方向 → 品质（硬/软）→ 强度 |
| **补光（Fill Light）** | 控制阴影深度，决定对比度 | 灯光比（低比=高调，高比=低调） |
| **背光（Rim/Back Light）** | 从后方照射，在边缘形成亮边，将主体从背景分离 | 方向、强度（可过曝）、品质 |

### 灯光比与情绪

| 灯光比 | 风格 | 适用场景 |
|--------|------|----------|
| 2:1 或更低 | 高调（High Key） | 时尚、喜剧、明亮氛围 |
| 3:1 ~ 4:1 | 中性 | 日常剧情、自然对话 |
| 8:1 或更高 | 低调（Low Key） | 悬疑、黑色电影、戏剧性 |

### AI 生成检查清单

- [ ] 主光是否从侧面/后方来（而非正面平光）？
- [ ] 补光是否足够控制阴影但不压平画面？
- [ ] 背光是否在主体边缘形成可见的分离亮边？
- [ ] 灯光比是否与场景情绪匹配？

## 二、硬光与软光

### 二元分类

灯光品质只有两种：**硬光和软光**。决定因素是**光源相对大小**，而非灯的功率。

| 属性 | 硬光 | 软光 |
|------|------|------|
| 阴影 | 清晰、尖锐 | 模糊、柔和甚至无阴影 |
| 原因 | 光线平行（小光源） | 光线从多方向到达（大光源） |
| 光源 | 直射阳光、聚光灯、裸灯泡 | 阴天天空、大面积柔光布、反射光 |
| 情感效果 | 冷硬、戏剧性、揭示质感 | 柔和、美化、减少瑕疵 |
| 适用 | 黑色电影、硬朗风格、质感表达 | 美化人物、柔情场景、时尚美容 |

### 制造软光的两种方法

1. **反射法**：硬光 → 大面积白色反射面（泡沫板、白墙） → 主体。反射面积越大，光越软。
2. **透射法**：硬光 → 柔光布（描图纸、丝绸） → 主体。柔光布应尽量靠近主体以最大化有效光源面积。

### AI 生成提示词

- 硬光：`hard directional light, sharp defined shadows, crisp texture, small light source`
- 软光：`soft diffused light, gentle shadow falloff, large light source, wraparound lighting`

## 三、灯光方向与情绪映射

### 主光方向决定情绪基调

| 主光方向 | 情绪效果 | 适用场景 |
|---------|---------|---------|
| 正面光（前光） | 扁平、无立体感 | 刻意使用：证件照、高调时尚 |
| 3/4前光 | 自然、中性 | 日常对话、新闻采访 |
| 侧光 | 戏剧性、明暗对比强烈 | 悬疑、黑色电影、揭示质感 |
| 3/4背光 | 神秘、深度感强 | 情感场景、深度叙事 |
| 背光 | 轮廓感、分离感、剪影 | 梦境、回忆、神圣感 |

### 核心原则

- 主光越从侧面/背面打来 → 主体深度和立体感越强
- **"显得暗"的正确方法**：不是降低曝光！而是将大部分灯光设为背光——画面曝光正确但观众看到的"主要是阴影面"，视觉感受上显得"暗"
- 正面平光是"最不立体且枯燥无聊的"布光方式

## 四、从后场打光与叠光技术

### 从后场打光原则

后场 = 远离摄影机的那一侧。从后场打光总比从前场（摄影机同侧）好：
- 产生好看的面部阴影，烘托脸部轮廓
- 自然地将主体从背景分离
- 避免"证件照"式扁平效果

### 叠光技术（Motivated Lighting）

当场景中有可见真实光源（台灯、窗户、壁炉）但不足以正确照亮演员时：

1. 在真实光源的照射方向上设置额外灯具
2. 额外灯具必须匹配真实光源的三个属性：**方向、品质（硬/软）、色温**
3. 通过减光器在真实光源亮度和演员曝光之间找到平衡

**关键**：叠光必须"重叠"——额外光与真实光方向一致，否则看起来不自然。

### AI 生成提示词

- 叠光：`motivated lighting from visible source, practical lamp illuminating character, warm tungsten spill`
- 后场打光：`key light from behind camera-opposite side, sculpting facial features, natural depth`

## 五、180度法则与空间连续性

### 基本规则

在两个角色之间定义一条**轴线（分割线）**，所有机位必须在轴线的同一侧。违反此法则 → 画面方向反转 → 观众困惑角色位置关系。

**定律**：不能让观众产生混淆——这是 180 度法则存在的唯一理由。

### 合法打破分割线的方法

| 方法 | 操作 |
|------|------|
| 中轴镜头 | 物体正对/远离摄影机 → 可过渡到另一侧 |
| 摄影机移动越线 | 在镜头中移动越过轴线，观众能理解空间关系改变 |
| 切到完全不同的画面 | 切到其他画面再切回来 → 可重新建立轴线 |
| 空间布局清晰 | 法庭、会议室等已知布局 → 可更灵活处理 |

### AI 多镜头生成中应用

生成对话场景的多镜头时，始终在轴线同侧规划所有机位。如果 A 角色在左侧看向右，B 角色必须在右侧看向左。

## 六、主场景拍摄法

### 核心策略

用于拍摄对话场景的最常用方法——"先建框架，再填细节"：

1. **先拍主镜头**：从头到尾拍摄完整场景（包含全部动作和对话），建立连续性基准
2. **再拍备选镜头**：过肩镜头 → 中景 → 特写，作为填充到主镜头框架中的"砖块"
3. **先完成一侧再换另一侧**：减少灯光和设备重复调整
4. **反应镜头与主镜头使用相同焦距和尺寸**：确保剪辑时人物大小一致

### 关键操作规则

- 人物进画面：从无人画面开始，再让演员走进
- 人物出画面：确保完全走出，多拍1-2秒
- 所有动作必须重叠拍摄（为剪辑提供安全余量）
- 主镜头是"合同"，后续镜头都是这个合同的"执行"

## 七、区域曝光系统

### 核心框架

源自安塞尔·亚当斯，将亮度范围分为 0-X 区（11级），每区相差一档曝光：

| 分区 | 含义 | 示例 |
|------|------|------|
| 第0区 | 纯黑，无细节 | 最深阴影 |
| 第V区 | 中灰（18%反射率） | 入射测光读数 = 灰卡读数 |
| 第X区 | 纯白，无细节 | 镜面高光 |
| 第VI区 | 亮色皮肤 | 典型肤色 |
| 第VII-VIII区 | 白色但保留细节 | 白衬衫、窗帘 |

### 两步操作法

1. **定位（Place）**：决定画面中关键物体的目标分区 → 调整曝光使它在目标位置
2. **跟随（Fall）**：其他物体按亮度差自动落入对应分区 → 如果跟随结果不理想 → 改变灯光比而非光圈

这两个操作将曝光从"正确曝光"变成了"**设计亮度关系**"的创意决策。

### AI 生成应用

描述图像亮度分布时可用分区思维：`the window curtain should read as Zone VIII (bright white with retained texture), the actor's face at Zone VI (natural skin tone)`

## 八、常见误区

1. **正面平光陷阱**：主光放在摄影机附近 = 最扁平无立体感的布光方式
2. **混淆光源功率与光质**：高功率灯+大面积柔光布 = 软光；功率只影响亮度
3. **用降低曝光代替改变灯光方向**：想要"阴暗感"→ 改用背光，不要降低曝光
4. **机械套用三灯位置**：三点布光是功能框架，不是"必须用三盏灯"
5. **跳轴无过渡**：直接从轴线一侧切到另一侧 → 观众瞬间失去空间方向感
6. **柔光布紧贴灯头**：应尽量靠近主体以最大化有效光源面积
