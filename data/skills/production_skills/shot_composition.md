# 大师级镜头构图技法

## 景别速查

| 景别 | 英文 | prompt关键词 |
|------|------|-------------|
| 远景 | Extreme Long Shot | wide shot, grand scene, epic scale, tiny figure in vast space |
| 全景 | Long Shot | full body shot, complete environment, establishing shot |
| 中全景 | Medium Long Shot | knee-up shot, body language visible, environmental context |
| 中景 | Medium Shot | waist-up shot, balanced composition, character and setting |
| 中近景 | Medium Close-up | chest-up shot, emotional expression, intimate but not too close |
| 近景 | Close-up | shoulder-up shot, facial detail, emotional intensity |
| 特写 | Extreme Close-up | extreme detail, single feature, eye/lip/hand macro |

## 角度速查

| 角度 | prompt关键词 | 叙事效果 |
|------|-------------|----------|
| 俯视/高角度 | high-angle shot, looking down, bird's eye view | 弱化主体，渺小感 |
| 仰视/低角度 | low-angle shot, looking up, heroic perspective | 强化主体，威严感 |
| 平视 | eye-level shot, neutral perspective | 中性平等 |
| 鸟瞰 | bird's-eye view, aerial shot, god perspective | 上帝视角，全局 |
| 荷兰角 | dutch angle, tilted camera, canted frame | 不安/失衡/紧张 |

## 构图法则速查

| 法则 | prompt关键词 |
|------|-------------|
| 三分法 | rule of thirds, subject placed at intersection points |
| 黄金分割 | golden ratio composition, 1.618 proportions |
| 对称构图 | symmetrical composition, perfectly balanced, centered |
| 引导线 | leading lines, converging lines, perspective guides |
| 景深对比 | shallow depth of field, bokeh background, subject isolation |
| 框架构图 | framing composition, foreground framing elements, window/doorway frame |

## Prompt构建公式

```
[景别] + [角度] + [构图法则] + [空间关系] + [主体描述] + [灯光氛围] + [画质约束]
```

示例：
```
Medium Close-up, eye-level shot, rule of thirds composition,
soft natural window light from left, shallow depth of field,
a young woman gazing out a rain-streaked window,
melancholic atmosphere, cinematic 8K quality
```

## 运镜技法速查

### 基础运镜（8种）

| 中文 | English | 技法 | 叙事效果 |
|------|---------|------|----------|
| 推 | Dolly In / Push-in | 相机移近被摄物 | 拉近距离、营造紧张/亲密感 |
| 拉 | Dolly Out / Pull-out | 相机远离被摄物 | 揭示环境、营造孤立/解脱感 |
| 摇 | Pan | 机位不动，水平旋转 | 追随动作、揭示空间 |
| 移 | Track / Truck | 机位横向移动 | 产生流动感、持续展现背景 |
| 跟 | Follow Shot | 镜头跟随移动主体 | 沉浸感、与角色同行 |
| 升 | Boom Up / Crane Up | 镜头上升 | 削弱主体、创造宏大/疏离感 |
| 降 | Boom Down / Crane Down | 镜头下降 | 强化主体、创造压迫/亲密感 |
| 甩 | Whip Pan | 快速水平旋转模糊转场 | 高能转场、制造迷失感 |

### 进阶技法（精选）

| 技法 | 技巧 | 效果 | 代表作品 |
|------|------|------|----------|
| **滑动变焦/Dolly Zoom** | 相机后退 + 镜头推近，主体大小不变但背景透视变形 | 眩晕、恐惧、心理崩溃 | 《迷魂记》《大白鲨》 |
| **荷兰角/Dutch Angle** | 画面水平线倾斜 | 不安、混乱、心理失衡 | 《第三人》《蝙蝠侠》 |
| **手持摄影/Handheld** | 无稳定器，刻意晃动 | 纪实感、紧迫感、身临其境 | 《谍影重重》《拆弹部队》 |
| **斯坦尼康/Steadicam** | 身体佩戴稳定器，平稳流畅 | 梦境般的空间连续性 | 《闪灵》丹尼骑三轮车 |
| **长镜头/Long Take** | 一个镜头内长时间连续拍摄 | 沉浸式真实感、不间断空间连续性 | 《好家伙》柯柏巴俱乐部3分钟 |
| **急推/Crash Zoom** | 快速从广角推到特写 | 突然强调、戏剧性揭示 | 《鬼玩人》系列 |
| **无人机航拍/Drone** | 无人机或直升机挂载相机 | 规模感、宏大叙事、俯视视角 | 《阿拉伯的劳伦斯》沙漠 |

### 运镜核心原则

- **无动机不运镜**：每一次相机移动必须服务于故事或情感。"因为看起来酷"不是理由
- **混淆变焦与推轨**：变焦改变焦距（透视保持平面），推轨改变位置（透视自然变化）
- **滥用手持脱敏观众**：好的手持与安静的静态镜头形成对比才有效
- **为炫技设计长镜头**：为展示而存在的长镜头破坏沉浸感

## 对话场景调度系统

### I/L/A 形态调度法（Steven D. Katz）

处理任意人数对话场景的三种几何形态：

| 形态 | 形状 | 适用场景 | 特征 |
|------|------|----------|------|
| **I 形态** | 两人面对面/并排直线 | 双人对话 | 最简单，视线方向清晰 |
| **L 形态** | 两人呈直角/L形 | 双人对话（更自然） | 身体朝向形成自然角度，更生活化 |
| **A 形态** | 三人呈三角形散开 | 三人及以上群戏 | 需要精心设计走位和机位覆盖 |

### 10种对话调度位置

| 位置 | 描述 | 情感暗示 |
|------|------|----------|
| 面对面 | 直接对峙 | 冲突、亲密、对抗 |
| 肩并肩 | 看向同方向 | 合作关系、同盟 |
| L形直角 | 身体呈90度 | 自然日常、非正式 |
| 背对背 | 互不看对方 | 疏离、冷战、决裂 |
| 一坐一站 | 高度差 | 权力关系、审视感 |
| 一人靠近一人退缩 | 动态距离 | 追逐/逃避 |
| 通过镜子/反射 | 间接对视 | 隐藏情感、间接沟通 |
| 一人面对镜头一人背对 | 不对称揭示 | 信息差、秘密 |
| 越过障碍物 | 桌子/栏杆/窗户隔开 | 阻碍、隔阂 |
| 围绕共同焦点 | 一起看向某物 | 共享注意力、同盟形成 |

### 对话场景的摄影机高度原则

- **摄影机高度 = 立场**：仰角给角色权威感，俯角给角色脆弱感
- **焦距递进**：随着情感接近，焦距逐渐从广角过渡到长焦
- **跟随即认同**：摄影机跟随哪个角色，观众就认同哪个角色

## 视点设计与观众认同

### 视点认同度谱系

每个镜头都有主观性程度，位于以下谱系上：

| 视点类型 | 主观程度 | 摄影机位置 | 观众体验 |
|----------|----------|-----------|----------|
| **全知视点** | 0%（完全客观） | 任何位置，无特定角色绑定 | 上帝视角，知道一切 |
| **第三人称有限** | ~40% | 跟随特定角色但不进入其视线 | 亲近但不代入 |
| **肩上视点** | ~70% | 角色肩膀后方 | 强烈的空间共感 |
| **主观视点/POV** | 100% | 完全替代角色眼睛 | 完全代入角色体验 |

### 视点选择原则

- 一场戏中视点不应频繁切换 → 观众需要稳定的认同锚点
- 重大揭示应使用视点降级（从有限→全知）制造信息差
- 情感高潮应使用视点升级（从有限→POV）增强代入感

## AI 镜头生成实用模板

### 情绪→运镜速查

| 情绪/叙事目标 | 推荐运镜组合 |
|---------------|-------------|
| 紧张/悬疑 | 慢推 + 限制画面 + 浅景深 + 长镜头 |
| 迷失/恐惧 | 荷兰角 + 滑动变焦 + 手持 + 不对称构图 |
| 孤独/疏离 | 后拉 + 升镜 + 留白 + 深焦 + 大远景 |
| 亲密/连接 | 缓慢推近 + 浅景深 + 手持呼吸感 |
| 宏大/敬畏 | 升镜 + 无人机拉远 + 广角 + 慢动作 |
| 混乱/紧迫 | 手持 + 急推 + 甩镜 + 快速剪辑 + 荷兰角 |
| 揭示/震惊 | 滑动变焦 + 快速推近 + 甩镜切静帧 |
| 安定/平稳 | 固定机位 + 水平线 + 对称构图 + 缓慢平移 |

### 五步镜头分析法

分析任何电影场景时的系统方法：
1. **识别镜头类型**：列出每种运镜（推、拉、摇、移、手持等）
2. **观察构图**：景别、角度、高度、镜头焦段
3. **描述运镜动机**：这个运动让观众感受到什么？
4. **联系导演签名**：这符合导演的一贯风格吗？
5. **解释为何有效**：技法如何在叙事/情感层面起作用？
